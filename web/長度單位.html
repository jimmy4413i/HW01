CSS 長度單位

CSS教學  >  CSS 長度單位

在 CSS ，長度單位可以分為兩大類：絕對 (absolute) 單位以及相對 (relative) 單位。

相對單位包括：

  • px: pixels
  • em: em
  • ex: x 字母的高度

    • 絕對單位包括：

  • in: 吋
  • cm: 公分
  • mm: 公厘
  • pt: points, 1 pt = 1/72 吋
  • pc: picas, 1 pc = 12 pt

    • 如果沒有註明單位，那預設的單位是 px。

    以下是每一種單位的例子：

    CSS 樣式 顯現結果
    p {font-size:9px;}

    文字大小為 9px。
    p {font-size:3em;}

    文字大小為 3em。
    p {font-size:3ex;}

    文字大小為 3ex。
    p {font-size:0.2in;}

    文字大小為 0.2in。
    p {font-size:0.5cm;}

    文字大小為 0.5cm。
    p {font-size:12mm;}

    文字大小為 12mm。
    p {font-size:24pt;}

    文字大小為 24pt。
    p {font-size:2pc;}

    文字大小為 2pc。

    在大部分的情況下，用相對長度單位會比較適合。這是因為，這樣子您能讓使用者依自己的喜好來
    加大或減小字體。如果您用絕對長度單位的話，使用者就無法改變字體的大小了。
